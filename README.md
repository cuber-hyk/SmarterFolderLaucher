# 智能文件夹启动器 (Smarter Folder Launcher)

一个基于 Electron + Vue 3 的智能文件夹快速访问工具，帮助您快速打开常用文件夹，提高工作效率。

## ✨ 功能特性

### 🚀 核心功能
- **快速启动**: 通过全局快捷键或托盘图标快速打开应用
- **文件夹管理**: 添加、删除、编辑常用文件夹，支持自定义颜色标识
- **智能搜索**: 支持中文拼音搜索，快速定位文件夹
- **系统集成**: 系统托盘常驻，开机自启动，无缝融入工作流

### 🧠 智能排序系统
- **优化算法**: 采用多因子加权的智能排序算法，更准确反映使用习惯
- **指数衰减**: 使用指数衰减函数实现更自然的时间权重计算
- **权重平衡**: 基础频率40% + 时间衰减30% + 累积分数30%的科学配比
- **平滑过渡**: 避免分数突变，提供稳定的排序体验
- **长期记忆**: 高频文件夹即使长时间未访问也能保持合理排名

### 🎯 多样化排序
- **🧠 智能排序**: 基于优化算法的智能排序，自动学习使用习惯
- **🔥 访问频率**: 按访问次数从高到低排序
- **⏰ 最近访问**: 按最后访问时间排序
- **📝 名称排序**: 按文件夹名称字母顺序排序
- **📅 创建时间**: 按添加到应用的时间排序

### 📊 使用统计
- **详细统计**: 记录文件夹访问次数和使用频率统计
- **可视化数据**: 在设置页面查看总文件夹数、总访问次数等统计信息
- **实时更新**: 访问记录实时更新，智能分数动态计算

### 🎨 界面体验
- **现代设计**: 毛玻璃效果界面，简洁美观
- **多主题支持**: 浅色、深色、蓝色、绿色、紫色、橙色等多种主题
- **响应式布局**: 支持窗口大小调整，适应不同使用场景
- **下拉选择器**: 优化的排序选择界面，节省空间提升体验

### ⚡ 性能优化
- **高性能**: 基于 SQLite 数据库，响应迅速
- **内存优化**: 高效的数据结构和算法实现
- **启动优化**: 快速启动，即开即用

## 🎯 快捷键

- `Ctrl + Alt + A`: 快速添加文件夹

## 📦 安装与运行

### 开发环境

1. 克隆项目
```bash
git clone <repository-url>
cd SmarterFolderLaucher
```

2. 安装依赖
```bash
npm install
# 或者使用 pnpm
pnpm install
```

3. 启动开发服务器
```bash
npm run dev
```

### 生产构建

```bash
# 构建应用
npm run build

# 仅构建渲染进程
npm run build:renderer

# 仅构建主进程
npm run build:main
```

## 🎮 使用方法

### 🚀 基础功能
1. **添加文件夹**: 点击底部的「添加」按钮或使用快捷键 `Ctrl + Alt + A`
2. **打开文件夹**: 点击文件夹项即可在文件管理器中打开
3. **搜索文件夹**: 在搜索框中输入关键词，支持中文拼音搜索
4. **管理文件夹**: 右键点击文件夹项可进行编辑或删除操作
5. **快速访问**: 使用 `Alt + F` 快捷键随时调出应用
6. **窗口调整**: 支持拖拽调整窗口大小，适应不同屏幕和使用场景

### 🧠 智能排序功能

#### 智能排序算法详解
智能排序采用多因子加权算法，综合考虑以下因素：

**算法公式**:
```
智能分数 = 访问频率权重(40%) + 时间衰减权重(30%) + 累积分数权重(30%)
```

**核心特性**:
- **指数衰减**: 使用 `exp(-天数差/30)` 实现自然的时间衰减
- **平滑过渡**: 避免分数突变，提供稳定的用户体验
- **长期记忆**: 高频使用的文件夹即使长时间未访问也能保持合理排名
- **自适应学习**: 系统会根据您的使用习惯自动调整文件夹顺序

#### 排序方式选择
在界面右上角的下拉选择器中可以切换不同的排序方式：

- **🧠 智能排序**: 基于优化算法的智能排序，推荐日常使用
- **🔥 访问频率**: 按访问次数从高到低排序，查看最常用文件夹
- **⏰ 最近访问**: 按最后访问时间排序，快速找到最近使用的文件夹
- **📝 名称排序**: 按文件夹名称字母顺序排序，便于查找特定文件夹
- **📅 创建时间**: 按添加到应用的时间排序，查看添加历史

#### 智能分数显示
- 智能分数会实时显示在文件夹右侧
- 分数越高，文件夹在智能排序中的位置越靠前
- 分数会根据使用情况动态更新

### 📊 使用统计

#### 统计信息查看
在设置页面可以查看详细的使用统计数据：
- **总文件夹数量**: 当前管理的文件夹总数
- **总访问次数**: 所有文件夹的累计访问次数
- **平均访问次数**: 每个文件夹的平均访问次数
- **已访问文件夹数量**: 至少被访问过一次的文件夹数量
- **使用频率分布**: 不同访问频率区间的文件夹分布情况

#### 访问记录显示
每个文件夹都会显示详细的访问统计信息：
- **访问次数**: 显示在文件夹名称下方，实时更新
- **最后访问时间**: 以相对时间显示（如"2小时前"、"昨天"等）
- **智能分数**: 在智能排序模式下显示当前分数
- **访问趋势**: 通过颜色深浅反映使用频率

#### 数据导出与备份
- 支持导出使用统计数据为CSV格式
- 自动备份访问记录，防止数据丢失
- 可选择性清除历史数据，重新开始统计

## 🔧 配置说明

应用数据存储在用户数据目录中：
- **Windows**: `%APPDATA%/smarter-folder-launcher/`
- **macOS**: `~/Library/Application Support/smarter-folder-launcher/`
- **Linux**: `~/.config/smarter-folder-launcher/`

### 配置文件说明
- `folders.db`: SQLite数据库文件，存储文件夹信息和访问记录
- `settings.json`: 应用设置文件，包含主题、快捷键等配置
- `logs/`: 应用日志目录，用于问题排查
